Description

TUDIVIEWER is a tool for the visualization of protein collections obtained from Thermo Protein Discoverer.

Features

  • Accepts .XLS, XLSX, CSV and TSV input files
  • Represents, in 2D gel-like format, LCMSMS identification data
  • Zooming of views
  • Diverse transforms (log2, square root, square) to equalize data in viewer.
  • Search by protein accession or name

Installation

Tested in Windows XP 32 bits and Windows 7 64 bits.

Requirements
If you work with TudiViewer source you need Python installed (tested with Python version 2.7.2) as well as the following dependencies:
  •     Matplotlib
  •     Pandas
  •     wxPython 2.8.11.2
  •     commons (from lpcsicuab googlecode repository)

From Installer
  1. Download TudiViewer_x.y_setup.exe windows installer from repository at the TudiViewer project download page.
  2. Get your Password for the installer.
  3. Double click on the installer and follow the Setup Wizard.
  4. Run TudiViewer by double-clicking on the TudiViewer short-cut in your desktop or from the START-PROGRAMS application folder created by the installer.
From Source
  1. Install Python and third party software indicated in Dependencies.
  2. Download TudiViewer source code from its Mercurial Source Repository.
  3. Download commons source code from its Mercurial Source Repository.
  4. Copy the folders in python site-packages or in another folder in the python path.
  5. Run TudiViewer by double-clicking on the kblast_main.pyw module.
Source Dependencies:
  • Python 2.6 or 2.7 (not tested with older versions)
  • wxPython 2.8.11.0 - 2.8.11.2
  • Pandas
  • commons (from LP CSIC/UAB Google Code repository)

Third-party program versions correspond to those used for the installer available here. Lower versions have not been tested, although they may also be fine.

Note for developpers: Installation from source requires some small quirks in pandas modules. (TO BE COMPLETED).
